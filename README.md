# CryptoPulse - Cryptocurrency Price Monitoring System

## Project Overview

CryptoPulse is a web-based cryptocurrency price monitoring system, prediction tool, and portfolio manager. It allows users to track real-time crypto prices, analyze technical indicators, forecast future prices using linear regression, and manage their investment portfolio.

This project was developed as an individual assignment for **TWT6223 - Web Techniques and Applications**.

## Live Demo

https://y1sh1en.github.io/CrytoPulse-System/

## Features

| Feature | Description |
|---------|-------------|
| Search & Track | Search cryptocurrencies by symbol (BTC, ETH, SOL, etc.) |
| Real-Time Price | Current price and 24h change from CoinGecko API |
| Interactive Chart | 30-day line chart with zoom (wheel) and pan (drag) |
| Technical Indicators | MA-7, MA-25, and RSI-14 with color coding |
| Price Prediction | Linear regression forecast for 7 days ahead |
| Portfolio Manager | Add, track, and delete holdings with P/L calculation |
| Drawing Tools | Trendline drawing with 6 color options |
| Annotation | Add text notes directly on the chart |
| Dark/Light Theme | Switch themes, preference saved in localStorage |
| Export CSV | Export portfolio data to CSV file |
| Auto-Refresh | Prices update automatically every 60 seconds |
| Error Handling | User-friendly messages for all error scenarios |

## Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Native styling (no frameworks), responsive design, dark/light theme
- **JavaScript (ES6+)** - All application logic
- **Chart.js** - Interactive chart rendering
- **chartjs-plugin-zoom** - Zoom and pan capabilities
- **CoinGecko API** - Real-time and historical cryptocurrency data
